ValueLengthRanking

@AsStringClassOption(includeIdentityHash = false, includeCompanion = false)
enum ValueLengthRanking : Enum<ValueLengthRanking>

ValueLengthRanking provides a somewhat arbitrary classification of value lengths, ranging from S to XXL, each with a rank and a lengthRange.

The lengthRanges fully cover the Int values from 0 to Int.MAX_VALUE, without overlap.

Parameters

rank

The relative weight associated with the ValueLengthRanking, ranging up from S to XXL

lengthRange

The range (min inclusive, max inclusive) associated with the ValueLengthRanking

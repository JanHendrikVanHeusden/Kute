propertyCoherentWithObject

val propertyCoherentWithObject: Boolean

Indicates whether the property's value can be resolved on the given object

  • true if no breaking incoherence was found between object and property

  • false if a wrong (incoherent) combination of object and property was passed to the constructor that would break property value resolution; in that case, the result will always be null

When false, a log message is issued on construction of the NamedProp, and the value will always be null, regardless on actual values of the object and the property.
